I needed to get the size of my database data and log files for several databases on a server, when I realized I didnâ€™t know how to group and sum this data in PowerShell.

dbatools has the Get-DbaDbFile command, which easily allowed me to collect information about the databases I needed.

Once I had this data, I wanted to group by the type of file and get the total size.Â  To start I piped the output to Select-Object to trim down the fields in the result set, then piped this output to the Group-Object, specifying the field I wanted to group by.

Get-DbaDbFile -SqlInstance $svr -Database CompressTest, AdventureWorks2017 |
Select-Object Database, TypeDescription, Size |
Group-Object TypeDescription

Now that the data is grouped you can aggregate the size property for the files, which is within the Group.Â  Weâ€™ll add an expression to the final Select-Object which uses the Measure-Object to sum the sizes.

Select-Object Database, TypeDescription, Size |
Group-Object TypeDescription |
Select-Object Name, @{l='Size';e={($_.Group.Size.MegaByte | Measure-Object -Sum).Sum}}
